# Habit Tracker Backend

### Installation

```bash
npm install
```

### Development

```bash
node server.js
## or
npm start
```

Hot reloading backend server with nodemon

```bash
npm run nodemon
```

### Dependencies

```
  - express
  - mongoose
```

### Dev Dependencies

```
  - cors
  - helmet
  - morgan
  - dotenv
  - nodemon
```
